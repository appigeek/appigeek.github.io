Site Appigeek — https://appigeek.github.io/ (racine)
====================================================

Le code est sur l’organisation appigeek. Il reste 2 actions sur GitHub :

1) Renommer le dépôt (obligatoire pour l’URL appigeek.github.io)
   https://github.com/appigeek/appigeek/settings
   → General → Repository name : appigeek.github.io
   → Rename

2) Activer GitHub Pages
   https://github.com/appigeek/appigeek.github.io/settings/pages
   (après le renommage)
   → Build and deployment : Deploy from a branch
   → Branch : main , dossier / (root)

Vérifier (après 2–5 min) :
  https://appigeek.github.io/
  https://appigeek.github.io/privacy.html
  https://appigeek.github.io/app-ads.txt

Fichiers à la racine : index.html, privacy.html, app-ads.txt
